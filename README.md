VitaDrop

VitaDrop is a real-time emergency healthcare Android application designed to connect **blood and organ donors with patients in critical need**.

Every year, many lives are lost because donors are not available at the right time. VitaDrop solves this by creating a **location-based emergency donor network** that works instantly during critical situations.

---

## Project Vision

> “No life should be lost due to lack of timely blood or organ availability.”

VitaDrop focuses on **speed, real-time response, and accessibility** to save lives.

---

## Features

- User Login & Registration (Firebase Authentication)
- Location-based donor search
- Emergency broadcast to nearby donors
- Hospital integration (Planned)
- Organ donor registration
- Live availability dashboard (Upcoming)

---

## Tech Stack

- **Language:** Kotlin  
- **UI:** Jetpack Compose  
- **Backend:** Firebase (Authentication + Realtime Database)  
- **Location Services:** GPS / Location APIs  
- **Notifications:** Push Notifications  
- **Version Control:** Git & GitHub  

---

## How It Works

1. User registers and logs in  
2. Location access is enabled  
3. In emergency:
   - App finds nearby donors  
   - Sends instant notifications  
4. Donors respond quickly  
5. Hospitals coordinate (future scope)
